3D Tic Tac Toe was created by Violet Graham. Inspired in part by the game of the same name for the Atari 2600.

The game was coded in JavaScript using Visual Studio Code. It uses p5.js a graphics library and addScreenPositionFunction library. Some of the graphics including the logo were created using Blender, Adobe Illustrator, and Photoshop.

The project's source code is available on GitHub at github.com/violetcraze/3D-tic-tac-toe. If you'd like to donate to help cover the maintenance costs of this site you can do so at ko-fi.com/violetcraze.
